# Video Stage

负责人：视频与 AI 教学开发者。

该目录负责 AI 教学页的双竖屏视频能力：

- 原手势舞播放器。
- 用户摄像头和跟练录制。
- 播放、暂停、慢放、回退和精确定位。
- 动作时间轴与视频切片。
- 动作提示、VLM 结果和骨骼信息的视频叠加层。
- 视频上传、下载和草稿衔接。

建议的内部目录：

- `components/`：播放器、摄像头、时间轴和叠加层。
- `hooks/`：MediaRecorder、播放同步和帧采集 Hook。
- `api/`：后端视频接口客户端。
- `types.ts`：该功能对外暴露的数据契约。

其他模块只能从 `index.ts` 引用公开能力。页面导航和整体布局仍由框架负责人维护。
